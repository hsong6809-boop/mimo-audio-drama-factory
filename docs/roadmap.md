# 开发路线图

## 总览

| 阶段 | 内容 | 周期 | 优先级 |
|------|------|------|--------|
| Phase 1 | 基础框架 + 剧本生成 | 1-2周 | P0 |
| Phase 2 | TTS 集成 + 情感控制 | 2-3周 | P0 |
| Phase 3 | 角色声纹系统 | 2-3周 | P1 |
| Phase 4 | 音频编排 + 后期 | 2周 | P1 |
| Phase 5 | 质量闭环 | 1-2周 | P2 |
| Phase 6 | 产品化 + 分发 | 2周 | P2 |

---

## Phase 1：基础框架 + 剧本生成（1-2周）

### 目标
搭建项目骨架，实现从"题材输入"到"结构化剧本输出"的完整链路。

### 任务清单

- [ ] 项目初始化
  - [ ] 目录结构创建
  - [ ] requirements.txt
  - [ ] 配置文件框架
  - [ ] 日志系统

- [ ] MiMo API 封装
  - [ ] 统一的 API 调用层
  - [ ] 重试/限流/错误处理
  - [ ] 流式输出支持

- [ ] 剧本创作 Agent
  - [ ] 角色设计（含声纹属性）
  - [ ] 剧情大纲生成
  - [ ] 逐集详细剧本生成
  - [ ] 输出结构化 JSON

- [ ] 基础 Pipeline
  - [ ] Orchestrator 主循环
  - [ ] 阶段间数据传递
  - [ ] 断点续传支持

### 交付物
- 可运行的命令行工具
- 示例输出：一个 3 集的仙侠剧本 JSON

---

## Phase 2：TTS 集成 + 情感控制（2-3周）

### 目标
将文本转化为带情感的语音，实现"同一句话不同情感说出来的效果不同"。

### 任务清单

- [ ] MiMo TTS 接口封装
  - [ ] 基础 TTS 调用
  - [ ] 参数控制（语速、音调、音量）
  - [ ] 批量合成支持

- [ ] 情感分析模块
  - [ ] 文本情感分类（8类）
  - [ ] 情感强度评估
  - [ ] 上下文情感连贯性

- [ ] 情感 TTS 引擎
  - [ ] 情感 → 参数映射表
  - [ ] 强度插值算法
  - [ ] 语速预设（slow/normal/fast）

- [ ] 音频片段管理
  - [ ] 片段命名规则
  - [ ] 元数据记录
  - [ ] 缓存机制

### 交付物
- 可合成带情感语音的 TTS 模块
- 演示：同一段文字，5 种不同情感的音频对比

---

## Phase 3：角色声纹系统（2-3周）

### 目标
确保同一角色在 100 集后声音仍然一致，不同角色声音可区分。

### 任务清单

- [ ] 声纹特征提取
  - [ ] librosa 特征提取（MFCC、音调、共振峰）
  - [ ] 特征向量化
  - [ ] 存储格式设计

- [ ] 声纹库管理
  - [ ] ChromaDB 集成
  - [ ] 相似度查询
  - [ ] 声纹更新策略

- [ ] 角色音色匹配
  - [ ] 新角色：自动推荐最接近的音色
  - [ ] 老角色：强制复用已有音色
  - [ ] 音色冲突检测

- [ ] 一致性校验
  - [ ] 合成后声纹比对
  - [ ] 偏离告警
  - [ ] 自动修正建议

### 交付物
- 声纹库管理系统
- 演示：同一角色跨 10 集的声纹一致性报告

---

## Phase 4：音频编排 + 后期（2周）

### 目标
将零散的音频片段编排成完整的有声剧，加入 BGM 和音效。

### 任务清单

- [ ] 音频拼接引擎
  - [ ] 段落间静音控制
  - [ ] 角色切换间隔
  - [ ] 场景过渡处理

- [ ] BGM 管理
  - [ ] BGM 分类库（按情绪/场景）
  - [ ] 智能匹配算法
  - [ ] 渐入渐出处理

- [ ] 音效管理
  - [ ] 音效库（脚步、开门、风声等）
  - [ ] 自动插入规则
  - [ ] 音量平衡

- [ ] 后期处理
  - [ ] 响度均衡（EBU R128）
  - [ ] 降噪处理
  - [ ] 均衡器调节
  - [ ] 片头片尾添加

### 交付物
- 完整的音频编排模块
- 演示：一集完整有声剧（含 BGM）

---

## Phase 5：质量闭环（1-2周）

### 目标
自动检测并修复质量问题，形成"生成→检测→修复"的闭环。

### 任务清单

- [ ] 文本质检
  - [ ] 字数检查
  - [ ] 特殊字符检测
  - [ ] 逻辑连贯性（MiMo 辅助）

- [ ] 音频质检
  - [ ] 静音检测
  - [ ] 音量异常检测
  - [ ] 时长匹配检查
  - [ ] 吞字/断句检测

- [ ] 角色一致性质检
  - [ ] 声纹比对
  - [ ] 角色串音检测
  - [ ] 情感一致性

- [ ] 自动修复
  - [ ] 问题分类与优先级
  - [ ] 自动重写/重合成
  - [ ] 人工审核队列

### 交付物
- 质检报告生成器
- 自动修复流水线

---
